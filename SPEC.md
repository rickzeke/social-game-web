# 社交轻游戏 - 真心话大冒险

## 1. Project Overview

- **项目名称**: 真心话大冒险 (Truth or Dare)
- **类型**: 社交互动小游戏
- **核心功能**: 朋友间互动的真心话大冒险游戏，支持多人轮流选择真心话或大冒险
- **目标用户**: 微信/抖音年轻用户群体

## 2. UI/UX Specification

### Layout Structure
- **单页应用**: 全屏滚动式交互
- **首页**: 开始游戏入口
- **游戏页**: 卡片式问答/挑战展示
- **分享页**: 炫耀成绩/邀请好友

### Visual Design

**Color Palette**
- 主色: `#FF6B9D` (浪漫粉红)
- 次色: `#8B5CF6` (神秘紫)
- 强调色: `#FFD93D` (活力黄)
- 背景: `#1A1A2E` (深邃夜空)
- 卡片背景: `#16213E` (深海蓝)
- 文字主色: `#FFFFFF`
- 文字次色: `#A0AEC0`

**Typography**
- 标题字体: "ZCOOL KuaiLe", cursive (可爱活泼)
- 正文字体: "Noto Sans SC", sans-serif
- 标题大小: 2.5rem (首页标题)
- 卡片文字: 1.5rem
- 按钮文字: 1.2rem

**Spacing**
- 页面内边距: 20px
- 卡片间距: 16px
- 按钮内边距: 16px 32px

**Visual Effects**
- 卡片: 玻璃拟态效果 (backdrop-filter: blur)
- 按钮: 渐变背景 + 发光阴影
- 页面切换: 淡入淡出 + 轻微上移
- 按钮点击: 缩放反馈 (scale: 0.95)
- 随机动画: 卡片出现时的弹跳效果
- 背景: 浮动气泡/星星粒子动画

### Components

**1. 首页**
- 大标题 + 副标题
- 游戏规则简介
- "开始游戏" 按钮 (渐变背景，悬停发光)
- 装饰性浮动图标

**2. 游戏卡片**
- 真心话/大冒险选择按钮
- 显示挑战内容
- 计时器 (可选)
- "下一个" 按钮
- 玩家轮次指示器

**3. 分享按钮**
- 微信分享图标
- 朋友圈分享图标
- 复制链接按钮

## 3. Functionality Specification

### Core Features
1. **选择模式**: 玩家选择"真心话"或"大冒险"
2. **随机挑战**: 从题库中随机抽取问题/任务
3. **玩家轮次**: 支持多人轮流游戏 (本地轮转)
4. **分享功能**: 
   - 微信分享 (Web Share API + 剪贴板)
   - 炫耀成绩卡片生成

### 题库内容
- 真心话: 30+ 有趣问题
- 大冒险: 30+ 有趣任务

### User Flow
1. 打开游戏 → 首页
2. 点击开始 → 进入游戏
3. 选择真心话/大冒险 → 显示挑战
4. 完成挑战 → 点击下一个
5. 分享成绩 → 邀请朋友
